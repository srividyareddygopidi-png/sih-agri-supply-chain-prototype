# SIH Agri Supply Chain Transparency Prototype

This project is a prototype developed for Smart India Hackathon 2025 to improve transparency in the agricultural supply chain.

## Features
- Farmer Login
- Batch Creation
- Warehouse Quality Check
- Trust Score System
- QR Code Generation for Batch Traceability
- Buyer Profiles
- IVR Service Simulation

## Tech Stack
- Python
- Streamlit
- QR Code Library

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
